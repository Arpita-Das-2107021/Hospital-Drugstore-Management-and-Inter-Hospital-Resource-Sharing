# Smart Disease Outbreak Prediction & Management System
### Based on 3 research papers — Bangladesh Pharmacy Network

---

## Project Structure

```
outbreak_project/
│
├── data/
│   ├── generate_data.py          ← Phase 1: generates random CSV data
│   ├── pharmacies.csv            ← EDIT THIS with real pharmacy GPS data
│   ├── sales.csv                 ← EDIT THIS with real sales data
│   ├── medicines.csv             ← EDIT THIS with real medicine list
│   ├── outbreaks_ground_truth.csv← for model evaluation
│   └── graph_dataset.json        ← auto-generated by build_graph.py
│
├── utils/
│   └── build_graph.py            ← Phase 2: builds GNN graph from CSV
│
├── models/
│   ├── train_stgnn.py            ← Phase 3+4: ST-GNN training
│   └── marl_dispatcher.py        ← Phase 5: MARL resource allocation
│
└── run_all.py                    ← Run the entire pipeline
```

## How to Run

### Step 0 — Install dependencies
```bash
pip install numpy
# For real GPU training later:
# pip install torch torch-geometric
```

### Step 1 — Run everything
```bash
python run_all.py
```

Or run each step separately:
```bash
python data/generate_data.py       # generate random data
python utils/build_graph.py        # build pharmacy graph
python models/train_stgnn.py       # train outbreak predictor
python models/marl_dispatcher.py   # run medicine dispatcher
```

## How to Use Real Data

### pharmacies.csv
Replace the 20 random pharmacies with your actual pharmacies:
```
pharmacy_id, name, upazila, lat, lon
PH001, Your Pharmacy Name, Joypurhat Sadar, 24.9012, 89.0234
...
```

### sales.csv
One row per medicine sold per day per pharmacy:
```
date, pharmacy_id, medicine_name, quantity_sold, upazila
2024-01-01, PH001, Paracetamol 500mg, 45, Joypurhat Sadar
...
```

### medicines.csv
```
medicine_name, base_daily_sales, outbreak_multiplier, signals_disease
Paracetamol 500mg, 30, 3.5, Fever/Flu
...
```
`signals_disease` must be one of: Fever/Flu | Diarrhea | Respiratory | Allergy/Fever | Normal

## What Each Paper Contributes

| Paper | What it adds to your system |
|-------|----------------------------|
| Paper 1 (VRE/Hospital GNN) | Heterogeneous graph structure, "living map" concept, temporal daily snapshots |
| Paper 2 (Evolve-DGN MARL) | Evolving GNN for changing graphs, MARL agents for resource dispatch, equity reward |
| Paper 3 (DGO-ST-GNN) | Spatiotemporal prediction from sales data, Grasshopper optimization, 97% accuracy target |
